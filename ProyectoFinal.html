1. Hardware

Conjunto de componentes físicos de un sistema informático, como procesador, memoria, disco duro y periféricos.

2. Software

Conjunto de programas y aplicaciones que permiten ejecutar diferentes tareas en un sistema informático.

3. CPU (Unidad Central de Procesamiento)

Es el cerebro del computador, encargado de ejecutar instrucciones y procesar datos.

4. Memoria RAM

Memoria de acceso aleatorio que almacena temporalmente datos e instrucciones mientras el computador está en uso.

5. Disco Duro

Dispositivo de almacenamiento permanente donde se guardan archivos, programas y el sistema operativo.

6. Tarjeta Madre

Placa principal donde se conectan los componentes del sistema, como CPU, RAM y dispositivos de almacenamiento.

7. Tarjeta Gráfica

Dispositivo encargado de procesar y renderizar gráficos en la pantalla del computador.

8. Fuente de Poder

Provee energía eléctrica a los diferentes componentes del computador.

9. Monitor

Dispositivo de salida que muestra la información visual del computador.

10. Teclado

Dispositivo de entrada que permite al usuario introducir datos mediante teclas.

11. Mouse

Dispositivo de entrada que permite mover el cursor y seleccionar elementos en la pantalla.

12. Impresora

Dispositivo de salida que permite obtener copias físicas de documentos digitales.

13. Escáner

Dispositivo que digitaliza documentos físicos para convertirlos en archivos digitales.

14. Sistema Operativo

Software que gestiona los recursos del hardware y permite la interacción entre usuario y computadora.

15. Navegador Web

Aplicación que permite acceder y navegar en internet.

16. Servidor

Computadora o sistema que proporciona servicios o recursos a otros equipos en una red.

17. Cliente Servidor

Dispositivo o aplicación que solicita servicios a un servidor dentro de una red.

18. Red LAN

Red de área local que conecta computadoras dentro de un espacio reducido como una oficina o casa.

19. Red WAN

Red de área amplia que conecta dispositivos en grandes distancias geográficas.

20. Internet

Red global que interconecta millones de dispositivos en todo el mundo para compartir información.

21. Intranet

Red privada interna utilizada dentro de organizaciones para compartir información y recursos.

22. Extranet

Red privada que permite a organizaciones compartir parte de su información con usuarios externos autorizados.

23. Dirección IP

Identificador numérico único que se asigna a cada dispositivo conectado a una red.

24. DNS

Sistema de nombres de dominio que traduce direcciones web en direcciones IP.

25. HTTP

Protocolo de transferencia de hipertexto utilizado para la comunicación en la web.

26. HTTPS

Versión segura de HTTP que utiliza cifrado SSL/TLS para proteger la comunicación.

27. HTML

Lenguaje de marcado utilizado para la creación y estructura de páginas web.

28. CSS

Lenguaje de diseño que se utiliza para dar estilo y formato a las páginas web.

29. JavaScript

Lenguaje de programación que permite añadir interactividad a las páginas web.

30. Base de Datos

Sistema organizado que almacena, gestiona y permite acceder a datos de manera eficiente.

31. SQL

Lenguaje de consulta estructurado utilizado para gestionar y manipular bases de datos.

32. Cloud Computing

Modelo de servicios que permite acceder a recursos informáticos a través de internet bajo demanda.

33. Inteligencia Artificial

Área de la informática que busca desarrollar sistemas capaces de imitar la inteligencia humana.

34. Machine Learning

Subcampo de la inteligencia artificial que permite a las máquinas aprender a partir de datos.

35. Big Data

Conjunto de datos extremadamente grandes que requieren tecnologías avanzadas para su análisis.

36. Internet de las Cosas (IoT)

Red de dispositivos físicos interconectados que recopilan e intercambian datos a través de internet.

37. Ciberseguridad

Conjunto de prácticas y tecnologías diseñadas para proteger sistemas informáticos y datos contra ataques.

38. Firewall

Sistema de seguridad que controla el tráfico de red entrante y saliente según reglas establecidas.

39. Antivirus

Software diseñado para detectar, prevenir y eliminar programas maliciosos.

40. Software de sistema

Permite al hardware y a otras aplicaciones funcionar correctamente. Incluye el sistema operativo (como Windows o macOS) y los controladores de dispositivos.

41. Conectividad

Capacidad de dispositivos para comunicarse entre sí.

42. Software de aplicación

Son programas diseñados para realizar tareas específicas para el usuario, como procesadores de texto, hojas de cálculo o navegadores web.

43. Software de programación

Son herramientas para desarrolladores que se utilizan para crear, modificar y ejecutar otros programas. Incluyen editores de texto, compiladores e intérpretes.

44. Arquitectura de software

Es el plano que define cómo se organizan e interconectan los componentes del software, como en la arquitectura de capas, que separa las responsabilidades en diferentes niveles.

45. Controladores

Permiten que el sistema operativo se comunique con el hardware específico, como impresoras o tarjetas de red.

46. Sistema operativo de IA

Integra la Inteligencia Artificial (IA) en su núcleo, permitiéndole aprender, adaptarse y mejorar con el tiempo mediante el análisis de datos e interacciones del usuario, en lugar de depender únicamente de reglas estáticas.

47. El sistema operativo Windows

Es una familia de sistemas operativos creada por Microsoft que gestiona el hardware y el software de una computadora, proporcionando una interfaz gráfica de usuario (GUI) para facilitar el uso de programas, documentos y la web.

48. Escritorio Windows

Es el espacio principal y el punto de partida que se ve al encender la computadora, donde se encuentran elementos como iconos, la barra de tareas, el botón de inicio y el fondo de pantalla.

49. Iconos

Son pequeños símbolos gráficos que representan programas, archivos o carpetas. Al hacer doble clic, se abren o ejecutan estas aplicaciones o contenidos.

50. Barra de tareas

Se encuentra generalmente en la parte inferior de la pantalla y permite acceder a programas abiertos, notificaciones y accesos rápidos.

51. Botón Inicio

Accede a todos los programas y configuraciones instaladas en el equipo.

52. Barra de Búsqueda

Permite buscar archivos, programas o información en internet de forma rápida.

53. Vista de Tareas

Un botón que muestra todas las ventanas abiertas y permite cambiar entre ellas.

54. Área de Notificaciones

Muestra la fecha y hora, los iconos del sistema, el volumen, el idioma y contiene el centro de actividades para ver notificaciones y alertas del sistema.

55. Ventanas

Son el área donde se ejecutan los programas y se realizan tareas, y pueden ser redimensionadas al arrastrar sus bordes o esquinas.

56. Menú de inicio

Es el punto de partida para acceder a todas las aplicaciones, archivos y configuraciones del sistema.

57. Cuadro de búsqueda

Permite buscar rápidamente archivos, aplicaciones, configuraciones del sistema y hasta buscar en internet.

58. Vista de tareas

Es una característica que permite organizar y cambiar entre las diferentes ventanas y aplicaciones abiertas.

59. Nuevo escritorio

Permite organizar y separar las diferentes tareas creando un nuevo espacio de trabajo limpio y personalizado.

60. Digitalización

Es un proceso tecnológico que permite convertir un documento en papel en una imagen digital gracias a procesos que conllevan técnicas fotoeléctricas o de escáner.

61. Carpetas en TIC

Son contenedores virtuales que permiten organizar archivos y otros directorios dentro del sistema operativo.

62. Archivos

Unidades de información almacenadas en la computadora que pueden contener texto, imágenes, audio, video o programas.

63. Panel de control

Herramienta de Windows que permite configurar el sistema operativo, el hardware y los programas instalados.

64. Explorador de archivos

Aplicación del sistema que facilita la navegación, búsqueda y gestión de carpetas y documentos almacenados en la computadora.

65. Papelera de reciclaje

Espacio donde se almacenan temporalmente los archivos eliminados antes de ser borrados de forma definitiva.

66. Actualizaciones del sistema

Mejoras y parches de seguridad que se descargan e instalan para mantener el sistema operativo protegido y eficiente.

67. Antivirus en ciberseguridad

Programa diseñado en Ciberseguridad para detectar, bloquear y eliminar virus y otras amenazas de seguridad en la computadora.

68. Extracción y análisis de certificados digitales

Revisión de certificados para validar autenticidad y detectar fraudes.

69. Copia de seguridad

Proceso de duplicar archivos importantes para evitar su pérdida en caso de fallos o ataques informáticos.

70. Nube informática

Servicio que permite almacenar, procesar y acceder a datos y programas a través de internet sin necesidad de tenerlos en el equipo físico.

71. Correo electrónico

Servicio digital que permite enviar y recibir mensajes, documentos e imágenes a través de internet.

72. Navegador web con IA

Un navegador web IA utiliza inteligencia artificial para automatizar tareas, personalizar la experiencia del usuario y proporcionar asistencia en tiempo real dentro de la navegación..

73. Buscador

Plataforma en línea que permite localizar información en internet mediante palabras clave.

74. Redes sociales

Sitios web o aplicaciones que facilitan la interacción, comunicación y el intercambio de contenidos entre personas.

75. Videollamadas

Comunicación en tiempo real mediante audio y video a través de internet.

76. Mensajería instantánea

Aplicaciones que permiten enviar y recibir mensajes de texto, voz o multimedia de manera inmediata.

77. Comercio electrónico

Compra y venta de bienes o servicios a través de internet en plataformas digitales.

78. Banca en línea

Servicio digital que permite realizar operaciones bancarias desde dispositivos conectados a internet.

79. La tecnología web engloba todas las herramientas

Lenguajes y sistemas que permiten crear, implementar y mantener sitios web y aplicaciones en internet, facilitando la comunicación y el intercambio de información a través de la world wide web.

80. Streaming

Transmisión de audio o video en tiempo real a través de internet sin necesidad de descargar los archivos.

81. Foros

Espacios en línea para discusiones y participación comunitaria donde los usuarios comparten ideas, preguntas y respuestas.

82. Análisis de tráfico cifrado (TLS) mediante metadatos

Evaluación de información accesible sin romper cifrados para detectar patrones.

83. Páginas de reservas

Sitios web que permiten a los usuarios reservar servicios como viajes, hospedajes o citas médicas en línea.

84. Sitios de membresía

Plataformas en línea que restringen el contenido a usuarios registrados o con suscripciones de pago.

85. Portafolios online

Sitios web diseñados para mostrar los trabajos, proyectos o logros profesionales de una persona o empresa.

86. Landing pages

Páginas diseñadas específicamente para convertir visitantes en clientes mediante una acción concreta como registrarse o comprar.

87. Web 1.0

Primera generación de la web, caracterizada por ser estática, de solo lectura y con poca interactividad.

88. Web 2.0

Segunda generación de la web, enfocada en la interacción, colaboración y participación activa de los usuarios.

89. Web 3.0

Conocida como la web semántica, se centra en el uso de inteligencia artificial para personalizar y comprender mejor la información.

90. Web 4.0

Proyección futura de la web, con inteligencia artificial avanzada, conectividad total e integración con la Internet de las Cosas.

91. Snapchat

Aplicación de mensajería instantánea enfocada en fotos y videos cortos llamados "snaps", que desaparecen tras ser vistos o en 24 horas si son publicados como historias.

92. Facebook

Red social creada en 2004 por Mark Zuckerberg que permite conectar con amigos, compartir fotos, videos, noticias y participar en comunidades en línea.

93. TikTok

Red social de origen chino creada por ByteDance que permite a los usuarios crear y compartir vídeos cortos con música y efectos.

94. WhatsApp

Aplicación de mensajería instantánea y red social que permite enviar mensajes de texto, voz, imágenes, videos, realizar llamadas y videollamadas.

95. YouTube

Plataforma propiedad de Google que permite a los usuarios subir, ver, comentar y compartir videos de todo tipo.

96. Twitter (ahora X)

Red social de microblogging que permite compartir mensajes cortos llamados tuits, además de fotos, videos y enlaces.

97. Spotify

Plataforma digital de música y pódcast que también integra funciones sociales como compartir playlists y seguir a otros usuarios.

98. Pinterest

Red social y motor de descubrimiento visual donde los usuarios guardan y comparten imágenes y videos organizados en tableros.

99. Reddit

Plataforma social basada en comunidades llamadas subreddits, donde los usuarios comparten, comentan y votan contenido.

100. Yahoo

Portal web que ofrecía correo electrónico, buscador, noticias y funciones sociales. También lanzó Yahoo Meme, similar a Twitter.

101. Endpoint

Es una URL específica que representa un recurso o una acción dentro de una API (Interfaz de Programación de Aplicaciones).

102. FastAPI

Es un framework moderno y rápido (de alto rendimiento) para construir APIs con Python, basado en tipado estático con Pedantic y Starlette para el core web.

103. JavaScript (ES6+)

Es la evolución moderna del lenguaje de programación JavaScript, basada en el estándar ECMAScript, que incluye la versión ECMAScript 2015 (ES6) y sus versiones posteriores.

104. TypeScript

Es un lenguaje de programación de código abierto de Microsoft que es un superconjunto de JavaScript.

105. Metaverso

Es un universo virtual tridimensional e inmersivo que evoluciona el internet actual, permitiendo a los usuarios interactuar social y económicamente a través de avatares en un ciberespacio compartido y persistente.

106. NFT (Token No Fungible)

Es un activo digital único e irrepetible, respaldado por la tecnología blockchain, que funciona como un certificado de autenticidad y propiedad para diversos tipos de contenido digital, desde obras de arte hasta vídeos o música.

107. Criptografía cuántica

Es un método de seguridad para cifrar y transmitir datos que se basa en las leyes de la mecánica cuántica, utilizando principios como la superposición y el entrelazamiento para garantizar la máxima confidencialidad.

108. Computación cuántica

Es un nuevo tipo de computación que usa los principios de la física cuántica, como la superposición y el entrelazamiento, para realizar cálculos muy complejos que superan las capacidades de las computadoras clásicas.

109. Edge computing

Es un modelo de computación distribuida que acerca el procesamiento de datos y el almacenamiento a la fuente donde se generan los datos.

110. 5G

Es la quinta generación de tecnología celular inalámbrica que ofrece velocidades de descarga y carga mucho más rápidas, una latencia ultrabaja (retraso mínimo) y la capacidad de conectar muchísimos dispositivos a la vez en comparación con las redes 4G.

111. 6G (en desarrollo)

Es la próxima generación de tecnología celular, actualmente en fase de investigación y desarrollo, que promete velocidades de hasta 1 TB/s, latencia casi nula, y una capacidad 50-100 veces superior al 5G.

112. Gemelos digitales vitaminados con IA

Un gemelo digital es una réplica virtual de un objeto, proceso o sistema del mundo real, que se actualiza en tiempo real con datos de sensores para permitir la monitorización, el análisis y la simulación del rendimiento.

112. Computación en la niebla (Fog computing)

Es una arquitectura de computación distribuida que acerca los recursos de procesamiento, almacenamiento y análisis de datos a los dispositivos y fuentes de datos, en lugar de enviarlos a una nube centralizada.

113. Biometría

Es un sistema de autenticación que utiliza características físicas (como huellas dactilares, iris o rostro) o de comportamiento (como voz o firma) únicas de una persona para verificar su identidad.

114. Redes Neuronales Artificiales

Son modelos computacionales inspirados en el cerebro humano, compuestos por nodos interconectados ("neuronas") organizados en capas, que procesan información para identificar patrones complejos en datos.

115. Análisis de registros de bases de datos (DB logs)

Revisión de logs de bases de datos para rastrear accesos o manipulaciones.

116. Deep learning

Técnica de inteligencia artificial y un subcampo del Machine Learning que utiliza redes neuronales artificiales con múltiples capas para aprender de grandes volúmenes de datos no estructurados, como imágenes o texto, a fin de identificar patrones complejos y realizar tareas de manera similar a los humanos.

117. (Machine learning)

Es un proceso que convierte activos físicos o intangibles (como una vivienda o datos sensibles) en unidades digitales llamadas tokens, que se registran en una blockchain, o en un proceso de IA, descompone un texto en unidades más pequeñas.

118. Contrato inteligente

Es un programa informático autoejecutable que se almacena en una cadena de bloques (blockchain) y que ejecuta automáticamente los términos de un acuerdo cuando se cumplen ciertas condiciones predefinidas.

119. Economía descentralizada (DeFi)

Es un sistema financiero abierto y colaborativo basado en la tecnología blockchain, que busca eliminar los intermediarios financieros tradicionales como bancos y corredores de bolsa.

120. Ciberfísico

Son aquellos que integran la computación, las redes de comunicación y los procesos físicos para monitorizar y controlar dichos procesos en tiempo real, conectando el mundo digital y el mundo real de forma inteligente.

121. Chatbot

Es un programa informático diseñado para simular conversaciones humanas, interactuando con los usuarios a través de texto o voz.

122. Automatización robótica de procesos (RPA)

Es una tecnología que usa robots de software para imitar las acciones humanas en tareas repetitivas y basadas en reglas.

123. Cómputo perimetral

Procesa datos cerca de su origen, minimizando la latencia y mejorando la velocidad de respuesta.

124. Procesamiento de lenguaje natural (NLP)

Es un campo de la inteligencia artificial y la lingüística computacional que permite a las computadoras comprender, interpretar y generar lenguaje humano a partir de texto y voz.

125. Reconocimiento facial

Es una forma de identificar o confirmar la identidad de una persona mediante una imagen de su rostro. Los sistemas de reconocimiento facial se pueden utilizar para identificar a las personas en tiempo real o para identificar a las personas en fotos o vídeos.

126. Algoritmo genético con IA

Técnica de optimización y búsqueda que imita el proceso de evolución biológica y la selección natural para encontrar la mejor solución a un problema complejo.

127. Realidad mixta

Es una tecnología que combina el mundo real con el digital, permitiendo a los usuarios interactuar con objetos virtuales y reales en tiempo real.

128. Industria 4.0

Es la transformación digital de los procesos industriales y de fabricación mediante la integración de tecnologías avanzadas como el Internet de las Cosas (IoT), la inteligencia artificial (IA), Big Data.

129. Microservicios

Son un estilo arquitectónico para construir aplicaciones como una colección de servicios pequeños, independientes y ligeramente acoplados, cada uno con su propio ámbito de responsabilidad y que se comunican a través de APIs bien definidas.

130. Contenedores de IA (Containers)

Son paquetes ligeros que encapsulan una aplicación y todas sus dependencias (código, bibliotecas, herramientas de sistema) para que pueda ejecutarse de manera consistente en cualquier entorno, desde el portátil del desarrollador hasta la nube.

131. Orquestación tecnológica

Es la automatización de la coordinación y gestión de múltiples sistemas informáticos, aplicaciones y servicios para ejecutar flujos de trabajo complejos.

132. Infraestructura como código (IaC)

Es la práctica de gestionar y aprovisionar recursos de TI a través de código, archivos de configuración y procesos automatizados en lugar de configuraciones manuales.

133. Computación sin servidor (Serverless)

Es un modelo de desarrollo de aplicaciones en el que se puede crear e implementar aplicaciones en una infraestructura de servidores administrados por terceros.

134. TIC

Tecnologías de la información y la comunicación, se refieren al conjunto de recursos y herramientas tecnológicas (hardware, software, redes) que permiten procesar, almacenar y transmitir información en formatos como voz, datos, texto, video e imágenes.

135. Criptomoneda con IA

Son un tipo de moneda digital que permite a las personas realizar pagos directos entre sí a través de un sistema en línea.

136. Criptoactivo

Es una representación digital de un valor o un derecho que se transfiere y almacena de forma electrónica, utilizando tecnología de registros distribuidos (como blockchain) y criptografía para garantizar su seguridad e integridad.

137. Minería de datos

Proceso de explorar grandes conjuntos de datos para descubrir patrones, tendencias y relaciones ocultas, utilizando técnicas estadísticas y de inteligencia artificial.

138. Privacidad diferencial

Es un sistema rigurosamente matemático para compartir información de un conjunto de datos de forma que se pueda analizar patrones grupales sin revelar información sobre ningún individuo.

139. Redes de malla (Mesh networks)

Sistema de red donde cada nodo se conecta con varios otros para mejorar la cobertura y la redundancia.

140. Vehículo autónomo

Automóvil que se conduce solo usando sensores, inteligencia artificial y sistemas de navegación.

141. Impresión biológica

Tecnología de impresión 3D para crear estructuras biológicas como tejidos u órganos.

142. Criptojacking

Uso no autorizado del dispositivo de alguien para minar criptomonedas.

143. Realidad extendida (XR)

Término que abarca realidad virtual, aumentada y mixta para experiencias inmersivas.

144. Plataformas Low-Code

Herramientas que permiten desarrollar aplicaciones con mínima programación manual.

145. Plataformas No-Code

Herramientas para crear aplicaciones sin necesidad de escribir código.

146. Infraestructura verde TIC

Tecnologías de TI diseñadas para ser sostenibles y reducir el impacto ambiental.

147. Sensores inteligentes

Dispositivos que recogen datos y procesan información para tomar decisiones autónomas.

148. Dispositivo wearable

Tecnología portátil que monitorea datos como salud, actividad o ubicación.

149. Computación cognitiva

Sistemas que simulan el pensamiento humano para procesar y aprender de datos.

150. Digital twin

Representación virtual de un objeto físico, sistema o proceso que se crea utilizando datos en tiempo real y técnicas de simulación para reflejar con precisión su comportamiento y rendimiento a lo largo de su ciclo de vida.

151. Ciberseguridad proactiva

Enfoque preventivo que detecta y detiene amenazas antes de que causen daño.

152. Quantum supremacy

Punto en que una computadora cuántica resuelve problemas imposibles para una clásica.

153. Data lake

Almacén centralizado que guarda grandes cantidades de datos en su formato original.

154. Dark web

Parte oculta de Internet accesible solo con herramientas especiales, donde se suelen realizar actividades ilegales.

155. Hashrate

Potencia de cómputo total usada para minar criptomonedas en una red blockchain.

156. Edge AI

Inteligencia artificial que procesa datos localmente en dispositivos, sin depender de la nube.

157. Federated learning

Técnica de aprendizaje automático donde los datos permanecen en el dispositivo.

158. Graphene tech

Tecnologías basadas en grafeno, un material ultra fino con propiedades excepcionales.

159. Neuromorphic computing

Computación inspirada en el cerebro humano para mejorar el procesamiento de datos.

160. Photonic chip

Chip que usa luz (fotones) en lugar de electricidad para transmitir información.

161. Spintronics

Tecnología que usa el spin de los electrones para almacenar y procesar datos.

162. Smart contract audit

Evaluación de contratos inteligentes para detectar fallos o vulnerabilidades.

163. Zero trust

Modelo de seguridad que no confía en ningún usuario o dispositivo por defecto.

164. Cyber resilience

Capacidad de una organización para resistir, responder y recuperarse de ciberataques.

165. Synthetic data

Datos generados artificialmente por algoritmos, usados para entrenar modelos de IA sin usar datos reales.

166. Tokenomics

Estudio de la economía interna de un token digital, incluyendo emisión, distribución, incentivos y gobernanza.

167. Heurística TIC

Aplicación de métodos heurísticos (atajos mentales o reglas empíricas) con el uso de las Tecnologías de la Información y la Comunicación (TIC) para resolver problemas en diversas áreas como la enseñanza de las matemáticas, la detección de virus en ciberseguridad y la optimización de algoritmos en inteligencia artificial.

168. Decentralized finance (DeFi)

Servicios financieros (préstamos, intercambios, seguros) que operan sin intermediarios, sobre blockchain.

169. Distributed ledger

Libro de contabilidad digital compartido entre múltiples nodos, que registra transacciones de forma segura y transparente.

170. Autonomous drone

Dron que opera sin intervención humana directa, usando sensores y algoritmos para navegación y tareas.

171. Hyper Ledger

Proyecto de software open-source para ledgers distribuidos y blockchain, usado sobre todo en entornos empresariales.

172. Multi-cloud

Uso de múltiples servicios de nube (cloud providers) para distribuir aplicaciones, datos o infraestructura.

173. Nanoelectronics
